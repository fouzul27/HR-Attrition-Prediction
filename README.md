# HR Attrition Prediction Dashboard

## Project Overview

This project analyzes employee attrition patterns using Python, Machine Learning, and Power BI. The goal is to identify factors influencing employee turnover and provide actionable recommendations for improving employee retention.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Power BI
* Jupyter Notebook

## Key Findings

* Overall Attrition Rate: 16.12%
* Monthly Income is the strongest predictor of attrition.
* Employees with low job satisfaction have higher attrition rates.
* Overtime significantly increases employee turnover.
* Employees with fewer years at the company are more likely to leave.

## Machine Learning Model

* Algorithm: Random Forest Classifier
* Objective: Predict whether an employee is likely to leave the organization.
